# **App Name**: Habit Hacker PWA

## Core Features:

- Add Habit Form: Form to create new habits with name, description, reminder message, and frequency.
- Habit List: Display list of user habits with details and a delete option with confirmation.
- Recurring Reminders: Send recurring notifications based on the user-defined frequency. This will be achieved by using set intervals.
- Adhkar Page: Display morning and evening Adhkar (Islamic supplications) in a tabbed interface.
- Summaries Page: Show a list of book summaries using an accordion design.
- Multilingual Support: Provide support for multiple languages (Arabic, English, French, Turkish, Indonesian, Persian) with RTL/LTR adaptation. Leverage a tool that intelligently adjusts reminders based on content, respecting cultural relevance across all supported languages.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to evoke calmness and focus.
- Background color: Light gray (#F5F5F5) for a clean, unobtrusive backdrop.
- Accent color: Muted green (#90EE90) for positive reinforcement and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif for its modern and friendly appearance. If longer text is anticipated, 'PT Sans' for the headlines and body copy.
- Use 'lucide-react' icons, ensuring they are clear, consistent, and contextually relevant.
- Utilize Tailwind CSS and ShadCN UI components to create a responsive layout that adapts to different screen sizes and orientations. Follow a grid-based system to maintain visual consistency across all pages.
- Subtle transitions and animations to enhance user experience when adding new habits or completing tasks.
